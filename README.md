A real-time grid-based visualizer built using C++ and SFML to demonstrate the efficiency and behavior of various pathfinding algorithms. This project was developed as a Complex Computing Problem (CCP) for the Object-Oriented Programming (OOP) course.

🚀 Features
Multiple Algorithms: Implementation of Dijkstra, BFS, and A*.

Strategy Pattern: A modular architecture that allows for seamless switching between different pathfinding strategies at runtime.

Interactive GUI: Users can place start/end points, draw walls, and watch the algorithm explore the grid in real-time.
